Position property values of "static", "relative", "absolute" and "fixed" Example

In this example, we have a div element with the class container that serves as the parent element. Inside this container, we have four div elements. Since we do not set any position propety for these boxes so they all have position "static".

In the below example, inside the container, we have four div elements, each with a different positioning method applied to it.

  • Red box: The static element has a position property set to static. This is the default position value, and it means that the element will flow in the normal document order, and its position cannot be adjusted using the top, bottom, left, or right properties. In this case, the element will appear in the top left corner of the parent container, with a red background color.
  • Blue box: The relative element has a position property set to relative. This means that the element's position can be adjusted using the top, bottom, left, or right properties, but it will still be positioned relative to its original position in the document flow. In this case, we have set top: 50px and left: 50px, so the element will be positioned 50 pixels down and 50 pixels to the right of its normal position. It has a blue background color.
  • Green box: The absolute element has a position property set to absolute. This means that the element is positioned relative to its nearest positioned ancestor element (got position relative), or if none exists, relative to the initial containing block (usually the viewport). In this case, we have set top: 100px and left: 100px, so the element will be positioned 100 pixels down and 100 pixels to the right of the container's top left corner. It has a green background color.
  • Purple box: The fixed element has a position property set to fixed. This means that the element is positioned relative to the viewport, and will stay in the same position even if the page is scrolled. In this case, we have set bottom: 150px and right: 50px, so the element will be positioned 150 pixels down and 50 pixels from the right edge of the viewport. It has a purple background color
