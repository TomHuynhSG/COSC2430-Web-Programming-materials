Example Media Query

This is an example of how to use media queries in CSS to change the color of a website.

Resize the browser window to see the color change.

Media query rules:

  • Extra small devices (phones, 600px and down): purple background color
  • Small devices (portrait tablets and large phones, 600px and up): darkblue background color
  • Medium devices (landscape tablets, 768px and up): darkgreen background color
  • Large devices (laptops/desktops, 992px and up): darkcyan background color
  • Extra large devices (large laptops and desktops, 1200px and up): darkred background color
